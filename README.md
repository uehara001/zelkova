# Zelkova
Zelkovaは競馬において馬や騎手、調教師などの能力値を表す指数の比率を操作することでユーザー独自の競馬指数を作る事の出来るツールです。
独自の競馬指数を短時間で感覚的に作ることが出来ます。
筋道立てて予想したい方や、感覚で予想したい方、初心者の方にも使いやすい作りになっています。
